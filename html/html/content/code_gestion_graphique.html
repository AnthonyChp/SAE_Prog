class fenetre_principale.AppGestion(donnees_spectateurs, donnees_concerts)[source]

Bases : QMainWindow

La classe AppGestion représente la fenêtre principale de l’application.

Cette fenêtre contient deux onglets, un pour la table des spectateurs et un pour la table des concerts. Chaque onglet comprend une table affichant les données de la table respective, ainsi que des boutons pour rafraîchir, ajouter, modifier et supprimer des entrées.

Variables:

  • donnees_spectateurs (DataSpectateurs) – L’objet gérant l’accès à la base de données des spectateurs.

  • donnees_concerts (DataConcerts) – L’objet gérant l’accès à la base de données des concerts.

class recup_concerts.DataConcerts[source]

Bases : QWidget

La classe DataConcerts permet de récupérer et d’exécuter des requêtes liées à la table des concerts dans la base de données.

Cette classe utilise la bibliothèque mysql.connector pour la connexion à la base de données MySQL.

Variables:

  • connexion_bdd (mysql.connector.connection.MySQLConnection) – L’objet de connexion à la base de données.

  • cursor (mysql.connector.cursor.MySQLCursor) – Le curseur utilisé pour exécuter des requêtes SQL.
